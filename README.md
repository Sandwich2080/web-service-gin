## 2. How to stop the Go web service

###  Step 1. Find the port that is using 
  
  (eg. Your server port is 8090)

  sudo Losf -i tcp:port

  eg. sudo Losf -i tcp:8090

###  Step 2. Find the PID and kill it

  sudo kill -9 PID

  eg. sudo kill -9 3210
